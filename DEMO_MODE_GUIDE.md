# Phoenix Pet 데모버전 안내

이 프로젝트는 로그인 없이 누구나 체험할 수 있는 데모 모드를 지원합니다.

## 데모 모드 특징
- 별도의 회원가입/로그인 없이 바로 펫 키우기, 건강습관(걸음수, 물 마시기, 스트레칭, 일찍 자기 등) 체험 가능
- 실제 데이터베이스에 저장되는 것이 아닌, 데모 전용 계정/토큰으로만 동작
- 모든 주요 기능(경험치, 레벨업, 진화, 상태 변화 등) 체험 가능
- 데모 모드에서 생성된 데이터는 실제 서비스에 영향을 주지 않음

## 사용 방법
1. 프로젝트를 실행하면 자동으로 데모 토큰이 localStorage에 저장되어 로그인 없이 바로 사용 가능
2. 홈 화면에서 다양한 건강습관 버튼을 눌러 펫을 성장시켜보세요!
3. 실제 회원가입/로그인 기능은 필요 시 별도 메뉴에서 이용 가능

## 데모 모드 구현 방식
- 프론트엔드(App.js)에서 앱 시작 시 localStorage에 임의의 데모 토큰(`demo-token`)을 자동 주입
- 백엔드에서는 데모 토큰이 오면 데모 계정으로 동작하도록 허용(실제 서비스와 분리)
- 데모 모드 관련 코드는 실제 서비스 코드와 명확히 분리되어 유지보수 용이

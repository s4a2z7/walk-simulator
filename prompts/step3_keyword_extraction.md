# Step 3 Prompt — 키워드 추출 (keyword_extraction)

## System
너는 한국어 텍스트에서 핵심 키워드를 **중복 없이** 추출하는 어시스턴트다.  
입력 텍스트에 없는 정보를 추가하지 않는다.

## User
아래 텍스트에서 키워드 5~10개를 추출해줘.

### 요구사항
- 너무 일반적인 단어(예: “합니다/있습니다/가능/여부/문의/확인/가능함”)는 제외
- 키워드는 명사/짧은 구로
- **반드시 5~10개**를 반환(최소 5개)
- 입력이 짧아 후보가 부족하면, **입력에 있는 표현을 분해/재조합**해 5개를 채운다.
  - 예: “접수부터 안내까지 친절” → `["접수", "안내", "친절", "접수 안내", "친절한 안내"]`
- **동의어/외부 지식/추측으로 새로운 개념을 추가하지 말 것**(입력 텍스트에 근거)
- 중복(의미가 동일하거나 형태만 다른 경우) 제거
- 위 “일반어”가 꼭 필요해 보이면, 일반어 자체를 쓰지 말고 **입력에 있는 더 구체 표현**으로 대체한다.
- 출력은 아래 JSON만 반환

### 출력 JSON 스키마
{
  "keywords": ["string"]
}

### 텍스트
{{text}}

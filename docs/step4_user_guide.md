## 1. 소개

**CareLog AI 챗봇**은 의료 서비스(병원/검진센터/약국) 예약과 안내를 돕는 대화형 AI 어시스턴트입니다.

### 주요 기능
- 예약 문의 응답
- 운영 시간/위치/준비물 안내
- 가격/절차 정보 제공
- 자연스러운 한국어 대화

---

## 2. 설치 및 실행

### 2.1 사전 요구사항
- Python 3.11 이상
- OpenAI API 키 (유료)
- 인터넷 연결

### 2.2 설치 단계

#### Step 1: 프로젝트 다운로드
```powershell
git clone <repository_url>
cd "CareLog AI"
```

#### Step 2: 의존성 설치
```powershell
# 기본 의존성
pip install -r requirements.txt

# 데모 전용 의존성
pip install -r requirements-demo.txt
```

#### Step 3: OpenAI API 키 설정
프로젝트 루트에 `.env` 파일 생성:
```
OPENAI_API_KEY=sk-proj-your-api-key-here
OPENAI_MODEL=gpt-4o-mini
```

**⚠️ 주의**: `.env` 파일은 절대 커밋하지 마세요!

#### Step 4: 데모 실행
```powershell
py -3 -m streamlit run demo\streamlit_app.py --server.headless=true
```

### 2.3 접속
브라우저에서 다음 URL로 접속:
```
http://localhost:8501/
```

## 3. 사용 방법

### 3.1 기본 사용

1. **브라우저 접속**: `http://localhost:8501/`
2. **메시지 입력**: 하단 입력창에 질문 입력
3. **전송**: Enter 또는 전송 아이콘 클릭
4. **응답 확인**: 챗봇 답변 확인

### 3.2 예시 질문

#### 예약 관련
```
오늘 건강검진 예약 가능한가요?
예약 취소는 어떻게 하나요?
예약 변경은 언제까지 가능한가요?
```

#### 안내 관련
```
영업시간이 어떻게 되나요?
주차 가능한가요?
위치가 어디인가요?
```

#### 준비 사항
```
건강검진 전에 뭘 준비해야 하나요?
공복이 필요한가요?
```

### 3.3 대화 초기화
- 사이드바(왼쪽) → "대화 초기화" 버튼 클릭
- 모든 대화 히스토리가 삭제됩니다

## 5. 문제 해결

### 5.1 "OPENAI_API_KEY가 없습니다" 오류

**원인**: `.env` 파일이 없거나 API 키가 잘못됨

**해결**:
1. 프로젝트 루트에 `.env` 파일 생성
2. 올바른 API 키 입력 확인
3. 서버 재시작

### 5.2 "Port 8501 already in use" 오류

**원인**: 8501 포트가 이미 사용 중

**해결**:
```powershell
# 포트 사용 프로세스 확인
netstat -ano | findstr ":8501"

# 해당 프로세스 종료 (PID 확인 후)
Stop-Process -Id <PID> -Force
```

### 5.3 챗봇이 응답하지 않음

**원인**:
- 인터넷 연결 끊김
- OpenAI API 서버 장애
- API 쿼터 초과

**해결**:
1. 인터넷 연결 확인
2. OpenAI 상태 페이지 확인 (status.openai.com)
3. API 사용량 및 결제 상태 확인

### 5.4 브라우저 접속 안 됨

**해결**:
1. 터미널에서 "You can now view your Streamlit app" 메시지 확인
2. 표시된 URL로 접속 시도
3. 방화벽 설정 확인

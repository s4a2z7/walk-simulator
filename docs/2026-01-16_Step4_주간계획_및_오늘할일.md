## 오늘 실행 커맨드(Windows/PowerShell)
```powershell
# 1) 데모 의존성 설치(권장: venv)
python -m venv .venv
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install -r requirements.txt
pip install -r requirements-demo.txt

# 2) OpenAI 키(.env)
# example.env를 .env로 복사 후 OPENAI_API_KEY 입력(커밋 금지)

# 3) 데모 실행
python .\demo\gradio_app.py
```

## 실행 확인(로컬)
- **Streamlit 챗봇 데모** (Gradio 호환 이슈로 전환)
- 접속: `http://localhost:8501/`
- 상태: `HTTP 200` 확인 완료
- 실행 커맨드: `py -3 -m streamlit run demo\streamlit_app.py --server.headless=true`
